### What is difference between edge and level sensitive circuits?
The difference is that level sensitive circuit stores the input value as long as its enable signal is high and an edge 
sensitive circuit stores the input value only when the anable transitions from low to high.
### Why is it important to declare initial state?
It is important to declare the initial state because the board has no way to know the state of any of the memory. This 
way we program the initial state so that right away the board knows what is in memory.
